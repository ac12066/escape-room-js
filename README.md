##  Descripción del Proyecto

Este proyecto es un Escape Room interactivo desarrollado con JavaScript, HTML5, CSS3 y Bootstrap. El objetivo es superar cinco niveles consecutivos para recuperar el acceso a una base de datos crítica para el funcionamiento de una ciudad inteligente.

###  Características Principales

- **5 Niveles Interactivos** - Cada nivel presenta un desafío único
- **Geolocalización** - Obtener ubicación actual del usuario
- **Canvas** - Dibujo de mapas simplificados
- **Cámara Web** - Captura de fotografías
- **Web Workers** - Procesamiento paralelo de datos
- **LocalStorage** - Almacenamiento de evidencias
